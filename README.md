# Portafolio Mairon Iturra

Este es mi portafolio personal, creado con React y Vite. Es una página única que muestra mis habilidades, experiencia y proyectos en SAP ABAP y BTP.

## Secciones

- **Hero**: Introducción y bienvenida
- **Sobre mí**: Breve descripción personal
- **Habilidades**: Lista de mis habilidades técnicas
- **Experiencia**: Mi experiencia laboral real
- **Proyectos SAP ABAP - BTP**: Proyectos destacados en SAP
- **Contacto**: Información de contacto

## Tecnologías utilizadas

- React
- Vite
- CSS

## Cómo ejecutar

1. Instala las dependencias:
   ```
   npm install
   ```

2. Ejecuta el servidor de desarrollo:
   ```
   npm run dev
   ```

3. Abre tu navegador en `http://localhost:5173`

## Construir para producción

```
npm run build
```

## Vista previa de producción

```
npm run preview
```
